# 个人评价

【有原则，冷静自持。

爱惜自己，珍惜家人朋友。

记忆力不是很好，过去的不愉快选择性忘记了。

懂得自己想要什么，为此努力和妥协。

不抱怨生活。

喜欢倾听和开导朋友。】

当然，这些东西并不是现在的我所拥有的，但却是我想成为的。

我对过去的自己一直不是很满意，如果能回到过去，我一定要给那个幼稚不成熟的自己一把掌，把他那所谓的迷茫统统打碎。

可正如那句话，【东隅已逝，桑榆非晚】，过去的已然成为历史，就好比在计算机操作系统中有一个概念叫做父子 shell，一个父 shell 可以创建多个子 shell，而如果在其中一个子 shell 中随意定义一个变量，其他的子 shell 是读取不到这个变量的。

我的意思是在不同的选择中我们就会有不同的状态，我们无法理解其他选择的自己，甚至是无法理解从前的选择的自己。我们能够做的也只有面对和珍惜。

所以说我们早已踏入风暴，身后已无退路。
